# Student Evaluation System (SET)

A comprehensive web-based system for collecting anonymous student evaluations of teachers to improve teaching quality.

## Features

✅ **Anonymous Evaluations** - Students can submit evaluations without identification
✅ **Survey Forms** - Customizable evaluation questions with rating scales
✅ **Automated Reports** - Generate performance summaries automatically
✅ **Faculty Performance Summaries** - Detailed analytics and insights
✅ **Admin Dashboard** - Manage teachers, students, and evaluations
✅ **REST API** - Full API for integration
✅ **Email Notifications** - Send reminders and reports
✅ **Advanced Analytics** - Trends, comparisons, and insights

## Technology Stack

- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+
- **Frontend**: HTML5, CSS3, JavaScript
- **Security**: Password hashing, CSRF protection, Session management

## Quick Start

1. Clone the repository
2. Import database schema: `mysql < database/schema.sql`
3. Configure database in `config/config.php`
4. Access at `http://localhost/student-evaluation-system`

## Default Credentials

- **Admin**: admin@system.com / admin123
- **Teacher**: teacher@system.com / teacher123
- **Student**: student@system.com / student123
